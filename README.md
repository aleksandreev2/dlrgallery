# DLR Gallery

DLR Gallery is a lightweight, modern Android photo gallery and basic image editor designed with lower-end devices such as Redmi 9 in mind.

## Product goals

- Fast photo grid backed by Android MediaStore
- Albums, favorites, viewer and settings tabs
- Smooth full-screen image viewing and zoom
- Non-destructive basic editing
- Low memory usage and restrained animations
- No ads, account or cloud dependency

## Current milestone

Milestone 0: project foundation and the four-tab Compose interface.

## Requirements

- Android Studio Quail 2 (or compatible)
- JDK 17
- Android SDK 37

## Run

1. Clone the repository.
2. Open the root folder in Android Studio.
3. Allow Gradle sync to finish.
4. Select an emulator or a connected Redmi 9.
5. Run the `app` configuration.
